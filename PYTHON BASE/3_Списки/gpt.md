```python
list.append(x): Добавляет элемент x в конец списка.
list.extend(iterable): Расширяет список, добавляя в конец все элементы из iterable.
list.insert(i, x): Вставляет элемент x в позицию i.
list.remove(x): Удаляет первый элемент, равный x.
list.pop([i]): Удаляет элемент в позиции i и возвращает его. Если индекс не указан, удаляет последний элемент.
list.clear(): Очищает список.
list.index(x[, start[, end]]): Возвращает индекс первого элемента, равного x.
list.count(x): Возвращает количество элементов, равных x.
list.sort(key=None, reverse=False): Сортирует список на месте.
list.reverse(): Разворачивает список на месте.
list.copy(): Возвращает копию списка.
```

```python
Вложенные списки:
Вложенные списки в Python - это списки, содержащие другие списки в качестве своих элементов.

nested_list = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

```

```python
Генераторы списков:
Генераторы списков предоставляют синтаксически удобный способ создания списков.

squares = [x**2 for x in range(10)]
```

```python
Слияние списков:
Слияние списков можно выполнить с использованием оператора +.

list1 = [1, 2, 3]
list2 = [4, 5, 6]
merged_list = list1 + list2
```

```python
Индексы и срезы:
Индексы используются для получения значений элементов списка.
Срезы используются для получения подсписка.

my_list = [10, 20, 30, 40, 50]

# Индексация
print(my_list[1])  # Вывод: 20

# Срезы
print(my_list[1:4])  # Вывод: [20, 30, 40]


```

```python

Примеры использования методов:

# append
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # Вывод: [1, 2, 3, 4]

# extend
my_list.extend([5, 6, 7])
print(my_list)  # Вывод: [1, 2, 3, 4, 5, 6, 7]

# insert
my_list.insert(1, 10)
print(my_list)  # Вывод: [1, 10, 2, 3, 4, 5, 6, 7]

# remove
my_list.remove(10)
print(my_list)  # Вывод: [1, 2, 3, 4, 5, 6, 7]

# pop
my_list.pop(1)
print(my_list)  # Вывод: [1, 3, 4, 5, 6, 7]

# и так далее...
```
