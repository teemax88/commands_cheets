## Методы списков

```python
list.append(x): Добавляет элемент x в конец списка.
list.extend(iterable): Расширяет список, добавляя в конец все элементы из iterable.
list.insert(i, x): Вставляет элемент x в позицию i.
list.remove(x): Удаляет первый элемент, равный x.
list.pop([i]): Удаляет элемент в позиции i и возвращает его. Если индекс не указан, удаляет последний элемент.
list.clear(): Очищает список.
list.index(x[, start[, end]]): Возвращает индекс первого элемента, равного x.
list.count(x): Возвращает количество элементов, равных x.
list.sort(key=None, reverse=False): Сортирует список на месте.
list.reverse(): Разворачивает список на месте.
list.copy(): Возвращает копию списка.
```

## Слияние списков
```python
# 5 7 вводим сначала длинну списков 1 и 2
# 2 8 8 16 19 вводим список 1
# 3 4 5 5 10 11 12 вводим список 2
# 2 3 4 5 5 8 8 10 11 12 16 19 должны получить такой список
n, m = 5, 7  # n,m = map(int, input().split()) если вводим с клавы
a = [2, 8, 8, 16, 19]  # a = list(map(int, input().split())) если вводим с клавы
b = [3, 4, 5, 5, 10, 11, 12]  # b = list(map(int, input().split())) если вводим с клавы
i = j = 0  # создали указатели индекса в списке
c = []  # создали пустой итоговый список
while i < n and j < m:  # пока указатели меньше чем длинна списка
    if a[i] < b[j]:  # переменная а с индексом[0] меньше b[0] ?
        c.append(a[i])  # добавили в переменную с значение a[i] (меньшее)
        i += 1  # сдвинули индекс меньшего на 1 выше
    else:
        c.append(b[j])  # добавили в переменную с значение b[j] (меньшее)
        j += 1  # сдвинули индекс меньшего на 1 выше
while i < n:  # если в списке (a) что-то осталось то запишем в (с)
    c.append(a[i])  # добавили в переменную с значение a[i] (меньшее)
    i += 1  # сдвинули индекс меньшего на 1 выше
while j < m:  # если в списке (b) что-то осталось то запишем в (с)
    c.append(b[j])  # добавили в переменную с значение b[j] (меньшее)
    j += 1  # сдвинули индекс меньшего на 1 выше
# print(c) печаем новый список так [2, 3, 4, 5, 5, 8, 8, 10, 11, 12, 16, 19]
print(*c)  # распакует список т.е будут отдельно цифры через пробел и без скобок = 2 3 4 5 5 8 8 10 11 12 16 19
# или так можно сделать print(*c) так :
# for i in c:
#     print(i, end=' ')
```

## Выражения-генераторы
+ *Выражения - генераторы не хранят в памяти все свои элементы, а выдают их по одному по мере надобности*
+ *Генератор - итератор, элементы которого можно итерировать только один раз*
+ *Итератор - объект, который поддерживает функцию next(). Помнит о том, какой элемент будет браться следующим*
+ *Итерируемый объект - объект, который предоставляет возможность обойти поочередно свои элементы.*
+ *Может быть преобразован к итератору.*

```python
# Генератор списка
a = [x ** 2 for x in range(1, 6)]
print(a)  # [1, 4, 9, 16, 25] создаст список из чисел от 1 до 5 в квадрате
```

```python
# Выражения-генераторы (как генератор списка только в круглых скобках)
b = (x ** 2 for x in range(1, 6))
print(next(b))  # 1 вызов итерируемого объекта сначала первый, потом последующий
print(next(b))  # 4
print(next(b))  # 9
print(next(b))  # 16
print(next(b))  # 25
# или можно обойти в цикле, но только 1 раз
for i in b:
    print(i)  # 1  4  9  16  25 (вывод будет только 1 раз или print(next(b)) или через цикл
print(sum(b))  # выведет сумму чисел в b только в первый раз прохода по ним(потом 0)
```
